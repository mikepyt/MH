Proyectos Destacados

Bot telegram notificacion licitaciones

En esta herramienta se integran tres elementos: scraping web, integración de API de Telegram y algoritmos de procesamiento de lenguaje natural, con el fin de facilitar la búsqueda de potenciales clientes para una pyme dedicada a la venta de equipos industriales y vehículos.

Se realiza scraping web en diferentes páginas de gobiernos estatales o empresas privadas que publican licitaciones abiertas para la adquisición de activos. Mediante algoritmos de procesamiento de lenguaje natural, se clasifica cada licitación abierta y, en los casos en los que la empresa puede participar, se envían dichas convocatorias por medio de un bot. Esto permite iniciar el proceso de prospección.

Automatización inventario

Se integra en una GUI para la carga y gestión de inventario en una pyme de venta de refacciones automotrices.

Para lograr la carga de inventario, se procesan facturas en formato PDF, donde la información contenida se estructura y se carga en una base de datos en AWS. Esta misma base de datos se utiliza en la herramienta para gestionar el movimiento de inventario por ventas. Además, se facilita la búsqueda de productos, ya que se segmentan por marca, automóvil y año.

Automatización de Reportes Financieros

Script que procesa datos financieros desde un sistema interno y genera gráficos interactivos para la gerencia para visualizacion en power BI.
